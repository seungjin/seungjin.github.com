AVOG (Assining and Visualizing Originas Geographically)2008 - 2010

  • Project name: Global DNA Database for Terrorism Related Identification(GDDTRI)
  • Funded by TSWG(http://www.tswg.gov) - United States Interagency program for research and development into counter-terrorism measures.
  • Constructed a database of Y chromosom and mitochondrial DNS haplotypes for 2000 convenience samples that were collected from 49 populations originating in the terrorist belt between East Africa and Indonesia. System is to use the database to infer the ethnic origin or genetic ancestry of suspected terrorist samples
  • AVOG is a rich web based application that visualizes DNA database and communicates with internal ancestry prediction engine(machine learning system)
  • Built with PHP ZendFramework for serverside layer. Built with Flex/Actionscript for rich web client service. Google/Yahoo Map API were used for presenting geographical data
  • Job title: Application Systems Analyst, Senior
  • Employer: Arizona Research Laboratories, University of Arizona
